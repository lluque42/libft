Detailed Description

Functions related to char **argv style arrays, that is, NULL terminated arrays of c-strings. Part of libft library.

void ft_free_strarr ( char **  strarr)

ft_free_strarr – Frees the memory allocated for an argv-style string array and its strings.

Frees the allocated memory for every string and for the array of an argv-style NULL terminated array of c-strings.

Parameters
[in]strarr- The argv-style NULL terminated array of c-strings.
Warning
A NULL pointer parameter, non-terminated array, or non-terminated c-strings will make this function crash.
Uses free().

ft_free_strarr_from – Frees the memory allocated for the strings of an argv-style string array from the element indicated by its index in the array.

Frees the allocated memory for every string starting at index 'from' of an argv-style NULL terminated array of c-strings. Every pointer of freed string is set to NULL. If from == 0, the behavior of this function is the same as ft_free_strarr(), that is, in this case it would also free the char** strarr.

Parameters
[in]strarr- The argv-style NULL terminated array of c-strings.
[in]from- The index in the array of the first string to free.
Warning
A NULL pointer parameter; non-terminated array; non-terminated c-strings; or invalid from value will make this function crash.
Uses free().

[in]fd- A valid file descriptor to which the argv-style NULL terminated array of c-strings will be output.
Warning
A NULL pointer parameter, non-terminated array, or non-terminated c-strings will make this function crash.

ft_remove_blanks_strarr – Remove strings composed only of blank characters from the string array.

Allocates memory for a new argv-style NULL-terminated array of strings consisting of newly allocated strings which are copies of those strings that are not composed solely of the blank_char.
This allows, for example when used on parsing, to remove those strings that are nothing more than a bunch of spaces.

Parameters
[in,out]strarr- The argv-style NULL terminated array of c-strings. It is the responsibility of the caller to free the newly allocated strarr.
[in]blank_char- The character to be considered as blank (e.g. ' ').
Warning
A NULL pointer parameter, non-terminated array, or non-terminated c-strings will make this function crash.
Uses malloc(), free().
